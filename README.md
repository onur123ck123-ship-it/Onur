# ÇeşmeAşk

Çeşme için sosyal check-in, swipe/match, AI öneri ve mini oyun prototipi.

## Canlı site
Netlify bağlantısı kurulduktan sonra bu repo otomatik deploy edilecek.

## Not
Bu sürüm hâlâ demo/prototiptir. Gerçek iki cihaz arası kullanıcı, check-in, match ve chat için sıradaki adım Supabase bağlantılı V5'tir.

## Batch plan
Login ekranı, renkli UI, mekan listesi, mesaj performansı ve oyun geliştirme işleri tek pakette toplanıyor.
